Beacon Borough API Project

A brief overview.This is a javascript project about both objects and for loops to apply to everyday New Yorker life. A project that I collaborated with Tony Ren, the purpose of the project was to use one of New York's many free APIs to create something that would loop through all the objects and produce a result. What I wanted the program to do was create a list a of Beacon programs within 10 - 15 miles of a location that someone puts in. Instead the code printed out a list of all the Beacon programs in a borough.

You might wonder, why the generalization? The problem was that for the longest time, the code would print either the number of programs but classify them as undefinded or it would print [Object, object] over and over again. Eventually it clicked and I used two empty sets to hold both the used objects (the beacon programs) and the unused objects and print the used objects. What I want to do next time is to incorporate more CSS to make the programs look more appealing to the eye and work properly with HTML.
